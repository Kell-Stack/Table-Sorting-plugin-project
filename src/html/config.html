Table Sorting Plugin


This plugin will allow you to multi-level sort your table


  1. 1. Choose the table you'd like to sort
    • - If you'd like to sort more than one table click the [+] icon

  2. 2. Choose the name of the column you'd like to sort, then your sort arrangement
    • - Primary sort will stick as your top level sort
    • - You can choose to have a secondary and tertiary sort that will stick, respectively
